Download Organizer

Auto-sorts your Downloads folder into tidy subfolders — the moment files land.

This package has both versions. Pick your computer:

🍎 Mac macOS folder

Lives in the menu bar (top-right of the screen).

  1. Open the macOS folder and unzip Download-Organizer-mac.zip.
  2. Drag Download Organizer.app into your Applications folder.
  3. First launch only: right-click the app → Open → Open.
  4. Click the 📥 icon in the menu bar for the menu & dashboard.
A normal double-click is blocked the first time — that's just macOS being cautious about apps from the internet. Right-click → Open gets past it.

🪟 Windows Windows folder

Lives in the system tray (bottom-right, near the clock).

  1. Install Python 3.10+ from python.org — tick Add Python to PATH.
  2. Open the Windows folder.
  3. Double-click build_windows.bat → makes dist\Download Organizer.exe.
    (Or run_windows.bat to just run it.)
  4. Right-click the tray icon for the menu; left-click opens the dashboard.
SmartScreen may warn the first run → click More info → Run anyway. Full details in WINDOWS-README.txt.

What it does

  • ⚡Watches Downloads and instantly moves each new file into the right subfolder — Media, Documents, Software, Archives, 3D-Models, Fonts, Dev…
  • 📊A modern dashboard shows what went where, with quick-open buttons.
  • 🔔A notification pops up each time a file is sorted.
  • 🔄Restart Service button (dashboard + menu) reloads your rules and restarts the watcher.
  • 🔒Sensitive / NFT / already-sorted folders are never touched.
  • ⚙️Edit the rules anytime via Edit Config in the menu.
Same sorting engine on both platforms — your Mac and PC organize files identically. Enjoy ⚡
